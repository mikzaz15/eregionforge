# MVP Scope

## MVP goal
Prove the core loop:
1. create project
2. add sources
3. compile wiki
4. inspect evidence-linked pages
5. ask questions against compiled knowledge
6. save artifacts

## In scope
- projects
- source ingestion for pasted text, markdown upload, URL placeholder, PDF placeholder
- source list and detail pages
- wiki compilation service
- wiki pages and revisions
- claims and evidence links
- ask mode over compiled knowledge
- artifacts
- linting basics

## Out of scope
- broad enterprise connectors
- collaboration and permissions complexity
- fine-tuning
- advanced graph visualization
- mobile-first UX
- full OCR and image pipelines

## MVP wow moment
A user imports sources and sees a real, navigable wiki appear with source summary pages, concept pages, open questions, and a memo generated from the compiled layer.
