# Agent Workflows

## Orchestrator
Routes work, sequences jobs, and ensures the product centers on compiled knowledge rather than freeform chat.

## Ingestion agent
Normalizes new source material and prepares it for extraction.

## Compiler agent
Produces canonical wiki pages and revisions from source-derived knowledge.

## Researcher agent
Answers user questions from the canonical layer first, then raw evidence if needed.

## Linter agent
Finds stale, unsupported, conflicting, or missing knowledge.

## Artifact writer agent
Turns research outputs into durable project assets.
